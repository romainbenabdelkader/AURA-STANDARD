# Contributing

AURA is an open public draft specification.

Contributions are welcome via:
- GitHub Issues
- Pull Requests

## Scope

Proposals should focus on:
- proof of origin
- signed provenance metadata
- interoperability with existing identifiers
- alignment with EU AI Act transparency requirements

Out of scope:
- watermarking
- fingerprinting
- content detection or similarity analysis
- DRM or rights enforcement systems

## How to contribute

- Open an Issue to discuss ideas or report problems
- Submit a Pull Request for concrete proposals
- Keep discussions technical, written, and specific
